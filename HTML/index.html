A Propos

Nous partons du principe qu'il n'est pas facile de prendre 9 semaines de votre temps pour apprendre le développement web ou les stratégies de marketing innovantes.

Nos formations de courte durées sont faites pour vous permettre de comprendre les outils de développement web et de mettre en place rapidement votre prototype pour votre start-up.

En Savoir Plus

Nos workshops

01. Créer votre protype web

Nous vous apprenons à créer vos landing pages, votre site internet Wordpress, votre site de e-commerce sur Shopify.

  • Les bases HTML, CSS ainsi que la librairie Bootstrap.
  • Développer de bonnes pratiques en web design.
  • Prototypage - UX/UI.
  • Les bases de la programmation Javascript côté Front-end.
02. Créez votre plan marketing à long terme sur les médias sociaux.

Nous vous apprenons à définir une stratégie digitale. Générer du traffic à partir des réseaux sociaux (Facebook & Instagram).

  • Faites de la publicité sur les médias sociaux.
  • Mesurez votre succès sur les médias sociaux.
  • Les pièges à éviter sur les réseaux sociaux.
